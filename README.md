# python-mineflayer
A python wrapper mineflayer node package based on websockets

## Usage
```py
import mineflayer

bot = mineflayer.Bot(**{
  "username": "PyBot",
  "host": "127.0.0.1",
  "port": 25565,
  "version": "1.16.4"
})
```
